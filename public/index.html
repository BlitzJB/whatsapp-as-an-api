How to Enter the Pairing Code in WhatsApp:

  1. You should receive a notification on your phone. Simply tap on it.
  2. Confirm that you are trying to link a device.
  3. Enter the pairing code shown above.
  4. If you don't receive a notification:

    • Go to Settings > Linked Devices.
    • Tap on "Link a Device".
    • At the bottom of the screen, tap "Link with phone number instead".
    • Enter the pairing code shown above.
